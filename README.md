# Прогнозування успішності студентів на основі навчальних та соціальних показників

**Курсова робота** з дисципліни *"Інтелектуальний аналіз даних"*

## Опис проекту

Цей проект присвячений розробці системи машинного навчання для прогнозування успішності студентів на основі їх навчальних та соціальних показників. Моделі використовують структуровані дані про студентів для передбачення їх фінальних оцінок та визначення факторів, що впливають на успіх в навчанні.

Проект реалізує повний ML pipeline: від завантаження та аналізу даних, через попередню обробку, до навчання моделей, їх оцінки та збереження результатів.

## Мета проекту

1. Розробити та порівняти кілька моделей машинного навчання для прогнозування успішності студентів
2. Провести розвідувальний аналіз (EDA) для виявлення закономірностей у даних
3. Підготувати дані та провести їх попередню обробку
4. Оцінити якість моделей за допомогою різних метрик
5. Визначити найбільш впливові фактори на успіх студентів

## Використані технології

### Мови програмування та їх інструменти
- **Python 3.x** - основна мова програмування
- **scikit-learn** - машинне навчання
- **pandas** - обробка та аналіз даних
- **numpy** - числові обчислення
- **matplotlib & seaborn** - візуалізація даних

### Моделі машинного навчання
1. **Logistic Regression** - лінійна модель для бінарної класифікації
2. **Decision Tree** - деревоподібна модель для класифікації
3. **Random Forest** - ансамбль дерев рішень

### Метрики оцінки
- Accuracy (точність)
- Precision (точність прогнозування)
- Recall (повнота)
- F1-Score
- ROC-AUC
- Confusion Matrix

## Встановлення та запуск

### Вимоги
- Python 3.7+
- pip (менеджер пакетів Python)

### Кроки встановлення

1. **Клонування репозиторію** (якщо це Git проект):
```bash
git clone <посилання_на_репозиторій>
cd student_success_prediction
```

2. **Створення віртуального середовища**:
```bash
python -m venv .venv
```

3. **Активація віртуального середовища**:
   - На Windows:
   ```bash
   .venv\Scripts\activate
   ```
   - На macOS/Linux:
   ```bash
   source .venv/bin/activate
   ```

4. **Встановлення залежностей**:
```bash
pip install -r requirements.txt
```

### Запуск проекту

```bash
python main.py
```

Це запустить повний pipeline:
1. Завантаження даних з `data/raw/student_data.csv`
2. Виконання розвідувального аналізу (EDA)
3. Попередня обробка даних
4. Навчання всіх моделей
5. Оцінка якості моделей
6. Збереження моделей та результатів

## Результати

Результати експериментів, включаючи метрики якості всіх моделей, зберігаються в:
- `reports/metrics/metrics.json` - числові результати
- `reports/figures/` - графіки та діаграми
- `models/` - навчені моделі

## Основні компоненти

### `src/data_loader.py`
Завантажує набір даних студентів з CSV файлу та готує його до подальшої обробки.

### `src/preprocessing.py`
Виконує очищення та трансформацію даних:
- Обробка відсутніх значень
- Кодування категоріальних змінних
- Масштабування числових ознак
- Розділення на навчальну та тестову вибірки

### `src/eda.py`
Проводить розвідувальний аналіз даних:
- Статистичні характеристики
- Кореляційний аналіз
- Розподіли ознак
- Виявлення викидів

### `src/modeling.py`
Ініціалізує набір моделей для навчання.

### `src/training.py`
Навчає всі моделі на тренувальній вибірці.

### `src/evaluation.py`
Оцінює якість моделей на тестовій вибірці за допомогою різних метрик.

### `src/persistence.py`
Зберігає навчені моделі та результати для подальшого використання.
